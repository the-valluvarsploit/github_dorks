🔭 .docker/.dockercfg
Docker Registry Authentication Data:
Configuration file used by the Docker client to authenticate to a Docker registry. It contains username, password, and email for the registry and is used to push and pull Docker images.

Look for:
1. auth

🔭 .mozilla/firefox/logins.json
Firefox Saved Password Collection
This file is a part of the password manager in the Firefox browser. It contains saved login credentials, such as usernames and passwords, that the user has saved while logging into websites.

Look for:
1. encryptedUsername
2. encryptedPassword

🔭 .ssh/id_rsa and .ssh/id_rsa.pub
RSA SSH Keys
These files are RSA private/public keys used in the Secure Shell (SSH) protocol. It is used to authenticate the user to a remote server, allowing them to establish an encrypted and secure connection.
